**An Agentic Personal Knowledge Management System (PKMS) engineered for the modern intellect.** NeuralFortress solves the cognitive bottleneck of the information age. Instead of manually sorting research papers, cloud architecture diagrams, and fragmented code snippets into rigid hierarchical folders, this system autonomously ingests, summarizes, and semantically maps your data. It acts as an intelligence synthesizer, giving you absolute recall over your entire digital footprint at the speed of thought.

Link: https://abhijeetkumarmonash.github.io/NeuralFortress/

## 🏗️ System Architecture

NeuralFortress is designed as a modern, decoupled monorepo, utilizing a high-performance stack optimized for AI integration and rapid data retrieval.

- **Frontend:** Vue 3 + Vite, styled with a custom Vibrant Glassmorphism UI (Bootstrap underlying).
- **Backend:** Python + FastAPI, providing asynchronous, high-speed endpoints for data processing.
- **Database:** Neon Serverless PostgreSQL, leveraging the `pgvector` extension for high-dimensional mathematical embeddings.
- **Intelligence Core:** RAG (Retrieval-Augmented Generation) pipeline driven by background LLM agents for auto-tagging and semantic search.

## ✨ Core Capabilities

* **Autonomous Ingestion:** Drop raw text, URLs, or code into the omni-input console. Background agents handle the sorting, summarizing, and categorization.
* **Vector Semantic Search:** Retrieve knowledge based on conceptual meaning rather than exact keyword matches.
* **Conversational Recall:** Chat directly with your localized knowledge base to synthesize answers from your past notes, perfect for deep-tech researchers and cloud architects.
* **Secure Local Execution:** Built with privacy and data sovereignty in mind.

## 🚀 Quick Start & Local Development

To run NeuralFortress locally, you will need to spin up both the frontend and backend servers. 

### 1. Initialize the Frontend (Vue UI)
Open a terminal instance and navigate to the frontend directory:

```bash
cd frontend
npm install
npm run dev
```
